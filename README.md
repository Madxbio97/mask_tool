# Dino Crisis 2: Mask & Background Matcher

Инструмент для сопоставления тайлов масок с фонами в игре Dino Crisis 2 и генерации увеличенных масок на основе обработанных нейросетями фонов.

## Возможности

- **Сопоставление (вкладка «Сопоставление»)**  
  Принимает директории с оригинальными фонами (≤320×240) и масками (≤256×256, с альфа-каналом).  
  Разбивает каждую маску на тайлы 16×16, содержащие непрозрачные пиксели, и ищет их на фонах.  
  Результат сохраняется в CSV-файл с колонками:  
  `mask, background, bg_x, bg_y, mask_x, mask_y`.

- **Выбор метода сравнения**  
  Доступны три алгоритма поиска тайлов:
  - `SQDIFF + refine` – базовый метод с уточнением позиции (по умолчанию).
  - `CCORR_NORMED + refine` – нормализованная корреляция + уточнение.
  - `Пирамидальный SQDIFF` – поиск на пирамиде изображений (быстрее при большом количестве тайлов).

- **Генерация увеличенных масок (вкладка «Генерация x4»)**  
  На вход подаются:
  - директория с увеличенными фонами (x4, имена совпадают с оригинальными);
  - CSV-файл, полученный на первом этапе;
  - директория с оригинальными масками (для извлечения альфа-канала).
  Программа вырезает из увеличенных фонов области 64×64 по координатам из CSV, совмещает их с увеличенной альфой и сохраняет новые маски (PNG с прозрачностью) с теми же именами.

- **Интерфейс**  
  Построен на PyQt5, все тяжёлые операции выполняются в отдельных потоках – интерфейс не зависает. Отображается прогресс и лог сообщений.
